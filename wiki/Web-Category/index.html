    • 개발자를 위한 웹기술 전반 : 웹 기술 전반에 대한 간단한 설명및 링크
      • MDN HTML : HTML 홈
      • MDN CSS : CSS 홈
        • CSS Reference : CSS 참고서
    • Web 개발 학습하기 : 따라해보는 예제 스타일의 웹 개발 자습서
      • HTML 자습서 : HTML 자습서
      • CSS 자습서 : CSS 자습서
      • Javascript 자습서 : Javascript 자습서
  • Google Fonts 홈
    • Get started : 구글 폰트 활용법, font effect 부분이 흥미로움..
    • Google Fonts 종류 : 골라쓰는 구글폰트
    • font developer api : 구글이 지원하는 웹폰트의 정보를 json 형식으로 return함
