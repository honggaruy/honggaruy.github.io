1. 개요

  • 인터넷에서 설치 파일을 다운로드 받을때 원본과 같은지 (data integrity) 확인할 수 있도록 hash값을 알려준다
    • e.g., Anaconda installer file hashes, docs.anaconda.com
    • Hashes for Anaconda3…exe
  • 각 OS별로 확인하는 cmd-line 툴이 있으므로 사용법을 알아보자
  • 참조 링크 :
    • Cyptographic hash verification, conda.io
    • sha256sum 명령어, John Grib 위키

2. OS별 확인 방법
