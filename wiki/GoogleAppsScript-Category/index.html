주요 사이트

  • Google Apps Script 홈
    • 안내: 전체적으로 뭐가 있는지 소개, 간단한 샘플코드 같은것도 있음
    • 참조: Api 분야별 자세한 명세서 모음
      • Manifests : Apps Script projects의 메타데이터 모음 파일의 스펙 명세, JSON 형식
      • G Suite Services : G Suite Service 앱에 bound된 class, method, property 명세
      • Advanced Google Services : 앱에 bound 되지 않고 별도로 연결해야 사용할수 있는 서비스 안내 (자세한 명세는 다른 사이트로 가야함)
      • Script Services : G Suite 특정앱에 bound 되지 않고, Google Script에서 기본적으로 사용할수 있는 built-in class, method, properties 명세
    • 샘플: 튜토리얼 모음?, 유튜브 비디오 모음, 따라해볼수 있는 Codelab 링크 등등이 있음
    • 지원: 도움 받을수 있는 방법 모음. stack overflow 태그 질문링크등등
    • Add-ons : 다른 동네 입구, G Suite Developer
    • Apps Script Api : 다른 동네 입구, Web Apps 형식으로 만들수 있는 동네인듯..
  • Google Drive API : 구글 Drive object의 API, 위의 GAS홈 » 참조 » Advanced Google Services 중의 하나임
    • 안내 : 전체 소개
    • 참조 : API 함수 소개
      • 엑셀 -> 구글 시트 변환 때문에 만든 script 에서 Drive.Files.insert를 사용하는데 version 2 내용임. v3에서 create로 바뀐듯함
    • 샘플 : 샘플 코드
    • 지원 : 도움 받을수 있는 방법 모음
    • Switch to v2 : 버전을 v2로 변경
  • Google Apps Script 대쉬보드 : 구글 앱스 스크립트 대쉬 보드, 요기서 스크립트를 직접 편집할 수 있다.
