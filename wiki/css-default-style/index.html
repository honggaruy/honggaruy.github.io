3. 소감

  • 원래 이 페이지를 작성하려던 의도는 Browser 기본 스타일이 적용되지 않을때 어떻게 적용시키나였다.
  • 이 글을 보고 head태그 안에 빈 script 태그를 넣으면 기본 스타일 적용되는 걸로 생각했는데…
  • 결론은 Content-Type 설정에 좌우 되는 것이었다.
  • charset: utf-8도 누락되면 인코딩 문제가 생기므로 꼭 추가한다
