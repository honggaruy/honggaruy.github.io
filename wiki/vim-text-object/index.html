개념

  • vim 명령을 동사 - 목적어의 개념으로 볼 때 text-objects란 목적어에 해당되는 부분
    • 즉 동사를 통해 내가 무언가 하고 싶은 부분을 정밀하게 선택한 부분을 text-objects라고 볼 수 있다

활용

  • WCX - 압축기 플러그인 (Packer plugins) 라는 텍스트가 있을때
    • “Packer plugins” 부분의 컬러를 변경하기 위해 mark 태그를 붙이고 싶다고 하자
    • ( 괄호 안의 텍스트는 i) text objects로 선택할수 있다
    • surround vim의 감싸기 operator인 ys를 사용하여
    • ( 괄요 안에 cursor를 두고
    • ysi)<mark>를 입력하면 ..
    • WCX - 압축기 플러그인 (<mark>Packer plugins</mark>) 와 같이 변경된다
