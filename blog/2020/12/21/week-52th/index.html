lofi 뮤직 ? code fi ?

  • Hi-Fi : High Fidelity의 약자.
    • 본래 음향용어로 원음에 충실하게 재현하는 고음질 음향 기기의 특성을 의미
  • Lo-fi : 그냥 Hi-Fi의 반대의미
    • 음질이 낮고 잡음이 많은 곡
    • 최근 고의적으로 이런 효과를 만들어 집어넣어 감성적인 효과를 노리는 곡들이 있다
    • 유튜브에 찾아 보면 많음
  • Code fi : Lo-fi 의 패러디로 코딩할 때 들으면 좋은 음악
    • 역시 유튜브를 찾아보면 나온다
